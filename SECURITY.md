# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in KasetPlus, please report it responsibly.

**DO NOT** open a public GitHub issue. Instead, email us at: **alessioiodiceuni@gmail.com**

Include:
- Description of the vulnerability
- Steps to reproduce (if applicable)
- Potential impact

We will respond within 48 hours and work on a fix.

## Supported Versions

| Version | Status |
|---------|--------|
| Latest  | ✅ Supported |
| Older   | ⚠️ Case by case |

Always use the latest version for the best security updates.

## Security Best Practices

- 🔄 Keep KasetPlus updated
- 🔐 Use a strong YouTube account password
- ⚠️ Only use trusted extensions/integrations
- 📢 Report suspicious behavior
